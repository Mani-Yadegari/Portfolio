# 💼 Mani Yadegari Portfolio

A modern and interactive personal portfolio built with **React** and **Vite** to showcase my projects, skills, and experience as a Front-End Developer.

## 🌐 Live Demo

🔗 https://maniydport.netlify.app

## ✨ Features

- Modern UI/UX
- Smooth page transitions & animations
- Responsive Design (Coming Soon)
- Project Showcase
- Contact Form (EmailJS)
- Download CV
- Fast performance with Vite

## 🛠 Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS3
- EmailJS
- Lucide React
- Lottie
- Git & GitHub

## 🚀 Installation

```bash
git clone https://github.com/Mani-Yadegari/Portfolio.git

cd Portfolio

npm install

npm run dev
```
